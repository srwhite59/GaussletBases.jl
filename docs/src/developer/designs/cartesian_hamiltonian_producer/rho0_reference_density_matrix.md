# Reference-Density-Matrix IDA Correction

Status: measurement-only authority for `HP-RHO0-REFDENS-AUDIT-01`.
Candidate future source IDs `HP-RHO0-REFDENS-FN-01` and
`HP-RHO0-REFDENS-ERI-01` are named for planning only and are not approved.

This page replaces the ambiguous row-gauge `rho0/Galerkin` line of thought as
the target formulation. The earlier `HP-RG-RHO0-GAL-AUDIT-01` probes remain
measurement evidence: they showed that `(J*w)/w`, `diag(J)`, and an IDA/MWG
density-proxy potential are different physical objects. They do not define the
production correction.

The corrected target is a fixed reference density matrix `P0`. The correction
must match both the exact reference energy and the exact reference first
derivative at that `P0`.

## Core Principle

Use a fixed reference density matrix, not only a scalar fitted density
`rho0(r)`.

For example, an atomic reference can be built by solving an atomic Hartree or
Hartree-Fock problem in the intended GTO supplement family, then placing the
resulting atomic density matrix on each nuclear center. The first approved
audit is Hartree-only, spin-summed, and measurement-only. HF exchange is a
later candidate lane.

The reference density must be representable by the current
protected-localized basis before it is used as a correction target. If the
reference uses GTO directions that are discarded, non-injectable, or outside
the compact main basis, the audit must report the loss and stop or mark the
result as approximate.

## Energy-Functional Algebra

Let `P_sigma` be spin density matrices in the final protected-localized working
basis. The corrected model is:

```text
E_new[P_alpha, P_beta] =
    E_app[P_alpha, P_beta]
  + sum_sigma Tr(P_sigma * Delta_F0_sigma)
  + C0
```

with

```text
Delta_F0_sigma = F_exact0_sigma[P0] - F_app0_sigma[P0]
```

where both exact and approximate Fock operators are evaluated at the same fixed
represented reference density `P0`.

The constant is required:

```text
C0 =
    E_exact0[P0]
  - E_app0[P0]
  - sum_sigma Tr(P0_sigma * Delta_F0_sigma)
```

Then the audit target is:

```text
E_new[P0] = E_exact0[P0]
dE_new/dP_sigma at P0 = F_exact0_sigma[P0]
```

Without `C0`, a Fock/orbital correction may still be useful, but the total
energy is not anchored.

## Hartree-Only First

The first audit is Hartree-only. For a spin-summed reference density

```text
P0 = P0_alpha + P0_beta
```

the exact Hartree operator in final basis functions `phi_i` is:

```text
J_exact[P0]_{ij} =
    sum_ab P0_ab * (phi_i phi_j | chi_a chi_b)
```

where `chi_a, chi_b` are reference GTO basis functions.

The exact Hartree reference energy is:

```text
E_exact0_H =
    1/2 * sum_abcd P0_ab * P0_cd * (chi_a chi_b | chi_c chi_d)
```

The approximate Hartree reference operator must be the operator generated by
the same IDA/MWG Hamiltonian convention that the solver uses for the same
represented `P0`. Do not invent a row-vector shortcut, `q0` shortcut, or
center-metadata substitution unless it is exactly the solver convention.

## HF Exchange Later

HF exchange is explicitly out of scope for the first audit. A later exchange
extension would need spin-resolved reference densities and exact exchange
integrals such as:

```text
(phi_i chi_a | chi_b phi_j)
```

in addition to the Hartree Coulomb integrals.

## Exact Integral Families

The Hartree-only audit must name whether the required exact integral data are
available or only approximated by probe code:

```text
(phi_i phi_j | chi_a chi_b)     final pair against reference GTO pair
(chi_a chi_b | chi_c chi_d)     reference self-energy
```

If compact RGs are present, `phi_i` includes them. The fixed reference-density
correction is a one-body operator in the final working basis, so compact RG
rows receive the exact reference Hartree/Fock correction too.

Injected/protected directions belong to the localized injected main basis.
They are not residual/MWG channels. Their approximate part is inherited from
the protected-localized site-order IDA convention.

Future source work will likely need a neutral exact mixed-ERI owner, not ad hoc
probe code hidden in Residual Gaussian files. Candidate `HP-RHO0-REFDENS-ERI-01`
is named only for that future design question.

## Reference Density Construction

The audit must record:

- atom and nuclear center;
- reference electron count per atom;
- reference basis name, `lmax`, contraction/uncontracted flags, and basisfile
  if any;
- spin treatment;
- density matrix convention and normalization;
- whether `P0` is full atom, core-only, or valence-screened;
- whether the reference was produced internally or by an external atomic
  calculation, with enough facts to reproduce it.

First target:

```text
Hartree-only closed-shell or spin-summed atomic reference
```

Cr measurement candidates may include an `[Ar]`-like core density, 18
electrons per Cr, if available. Other screen counts such as 10, 18, or 22 are
measurement variants only.

## Representability Check

Before building the correction, ask:

```text
Is P0 represented by the current protected-localized basis?
```

Required representability diagnostics:

- reference occupied orbital overlap with final protected-localized space;
- density trace loss;
- projected-reference density norm loss;
- per-owner and per-angular-channel lost components;
- whether loss is due to broad non-injectable directions.

If a good reference direction fails representability in the current main basis,
that is an insufficient-main-basis diagnostic. Do not convert it into a broad
MWG residual channel.

## Approximate Reference Fock

`F_app[P0]` must be the Fock/Hartree operator generated by the current
approximate Hamiltonian convention for the same represented reference density.

Do not define `F_app[P0]` by:

- row action `(J*w)/w`;
- `diag(J)`;
- center metadata;
- an invented row-vector subtraction;
- direct `C' V C` interaction rotation.

The audit must include a finite-difference check where feasible:

```text
(E_app[P0 + eps*dP] - E_app[P0 - eps*dP]) / (2*eps)
    ~= Tr(dP * F_app0)
```

If this fails, the approximate reference operator is not the solver's Fock
operator.

## Exact Reference Fock

Similarly, validate the exact reference operator by finite differences of the
exact reference energy where feasible:

```text
(E_exact[P0 + eps*dP] - E_exact[P0 - eps*dP]) / (2*eps)
    ~= Tr(dP * F_exact0)
```

For Hartree-only probes this should be practical because the exact reference
energy is the Coulomb self-energy.

## Do Not Confuse

- scalar `rho0` with a reference density matrix `P0`;
- `q0` row coordinates with a GTO density matrix;
- row action `(J*w)/w` with the IDA/MWG density-proxy potential;
- `diag(J)` with the IDA/MWG density-proxy potential;
- `F_app[P0]` with an invented row-vector subtraction;
- one-body operator transforms with two-index IDA interaction transforms;
- direct `C' V C` rotation with the protected-localized inherited-site IDA
  convention;
- injected directions with residual MWG channels;
- compact RGs with broad injected GTOs;
- total-energy correction with Fock-only correction.

Injected directions replace a subspace of the compact main basis. They are not
added to `G`, and only true compact residual directions get residual-GTO/MWG
channels.
