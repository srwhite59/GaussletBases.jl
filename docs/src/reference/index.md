# Reference overview

This is the first curated API-reference slice for `GaussletBases.jl`.

It is intentionally built from selected `@docs` blocks rather than a full
automatic export dump. The goal of this first pass is to document the main
entry points that a user is most likely to call directly.

Use these topic pages:

- [Bases and mappings](bases_and_mappings.md)
- [Operators and diagnostics](operators_and_diagnostics.md)
- [Atomic and ordinary workflows](atomic_and_ordinary.md)
- [Export layer](export.md)

If you are learning the package, start in the [Manual](../manual/index.md)
first. Come here when you want the API meaning of a specific object or
function.

The narrative pages still carry the workflow and interpretation. The reference
pages answer the more standard API questions: what a function or object is,
what it returns, and how it fits into the current model.
